# StrategyForge Agent Demo

StrategyForge Agent 是一个面向 FLOW AI 投资交易 Agent 黑客松的可运行 Demo。

它的目标不是直接给出“稳赚策略”，而是把用户的一句话投资想法转成结构化策略，并自动补全风控、基准对照和回测报告。

## 功能

- 读取本地 `DATASET` 文件夹中的 A 股历史日线数据
- 先通过 Agent 对话追问用户想法、标的、周期、风险偏好和研究目标
- 根据用户想法自动识别策略方向：趋势跟踪、均值回归、布林带反转、多策略投票、基础模板
- 生成策略卡：市场假设、入场、退出、风控、基准
- 自动补全：仓位、止损、波动率过滤、交易成本、买入持有基准、候选策略
- 多候选回测：用户想法基础版、Agent 风控增强版、稳健多策略投票版、反向风格备选
- 输出指标：累计收益、基准收益、超额收益、最大回撤、夏普比率、交易次数、胜率、持仓天数占比
- 策略评分：收益分、回撤分、稳定性分、成本分、目标适配分、综合评分
- 可视化：净值曲线、买卖点、回撤曲线、评分条形图
- 生成自然语言研究报告

## 运行

在项目根目录执行：

```powershell
cd demo
pip install -r requirements.txt
streamlit run app.py
```

如果只想测试策略和回测逻辑：

```powershell
python strategyforge.py
```

## Demo 讲法

1. 用户输入一句模糊想法，例如“我觉得创业板 ETF 最近跌多了可能会反弹，但不想承受太大回撤。”
2. Agent 不直接给结论，而是逐轮追问标的、持有周期、风险偏好和研究目标，并说明为什么要问。
3. Agent 把回答提取成结构化策略卡：市场假设、入场规则、退出规则、风控规则。
4. 系统生成多套候选策略，而不是只给一个答案：基础版、风控增强版、多策略投票版、反向风格备选。
5. 回测所有候选策略，并与买入持有基准比较。
6. 用户选择单个方案查看买卖点、回撤曲线和评分条形图。
7. Agent 输出自然语言研究报告，说明哪套策略更适合当前目标、风险在哪里、下一轮怎么优化。

## 边界

本项目仅用于策略研究、模拟回测和黑客松技术展示，不构成投资建议，不承诺收益。LLM/Agent 不直接执行真实交易；买卖信号、回测指标和风险计算均由确定性代码完成。
